# It's Battleship Y'all!

## Features

- Drag & Drop Ship Placement UI
- Somewhat Intelligent Bot AI
- Three Gamemodes
  - Bot vs Bot
  - Player vs Bot
  - Player vs Player (local)

## Improvement possibilities
